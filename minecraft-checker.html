Frequently Asked Questions

What is a UUID?

A UUID (Universally Unique Identifier) is a permanent ID for a Minecraft account. While a player can change their username, their UUID never changes. This makes it the most reliable way for servers to track players for permissions, inventory, and bans.

How Does This Profile Search Work?

Our tool uses the official Mojang API, the same one used by the game itself. We first use the username to find the account's UUID. Then, we use that UUID to fetch the name history and to generate a real-time render of the player's current skin using the Crafatar API.
